# Exclusive E-Commerce Website

## Features
- User Registration & Login
- Product Listing
- Add to Cart
- Remove from Cart
- Checkout System
- Payment Options (Card, UPI, Net Banking)
- Order History
- Admin Panel (Products + Orders)

## Technologies Used
- ASP.NET Core MVC
- Entity Framework Core
- SQL Server (SSMS)
- Bootstrap
